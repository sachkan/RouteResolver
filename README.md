# routeResolver
Handling many routes




Require :
  Use Composer to install Autoload PSR-4: Autoloader https://www.php-fig.org/psr/psr-4/
in composer.json change "App\\" to specific your project folder, ex: "app/" my project folder.

    "autoload" : {
        "psr-4" : {
            "App\\" : "app/"
        }
    }
